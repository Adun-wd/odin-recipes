# odins recipes

This is a project from the odin projects curriculum, where I am building a simple recipe website using HTML

## About this Project

The goal is to create a website with a home page  using several recipes, and individual pages for each recipe ingredients and instruction. This project is focus on HTML fundamentals.

### Skills I will diplay

- Structuring a webpage using semantic HTML
- Creating and linking multiple HTML pages
- Using Git and GitHub for version control (cloning, staging, committing, pushing)
- Working with a local development environment (VS Code, terminal)
